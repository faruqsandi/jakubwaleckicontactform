About This Page

This is a hidden testing page for Huey background task functionality. It allows you to:

  • Trigger dummy tasks: Test basic background task execution
  • Trigger form detection tasks: Simulate background form detection work
  • View recent tasks: See the last 100 tasks executed by Huey
  • Monitor task status: Check if tasks completed successfully
Important Notes:
  • This page is not linked from anywhere in the application
  • Huey consumer must be running separately for tasks to execute
  • Tasks are stored in SQLite database: huey_tasks.db
  • Start the Huey consumer with: huey_consumer huey_config.huey
